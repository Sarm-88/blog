## Tips
 - Find .md link and convert it to .html link

    ```
    %s/\[\(.*\)\](src\/\(.*\)\.md)/[\1](docs\/\2.html)
    ```

 - Markdown goto link: ```gf```

 - Goto previous buffer: ```Ctrl-^```
